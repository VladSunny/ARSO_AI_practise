## Что когда использовать


| Данные                     | Подход                                                 | Ноутбук                              |
| -------------------------- | ------------------------------------------------------ | ------------------------------------ |
| Таблица (числа, категории) | CatBoost / XGBoost                                     | [tabular.ipynb](tabular.ipynb)       |
| Таблица + нужен sklearn    | Линейные модели, KNN, SVM, деревья, кластеризация, PCA | [classic_ml.ipynb](classic_ml.ipynb) |
| Нужна нейросеть на PyTorch | FC-сеть + train loop                                   | [dl_pytorch.ipynb](dl_pytorch.ipynb) |
| Изображения                | CNN / pretrained ResNet / fine-tuning                  | [cv.ipynb](cv.ipynb)                 |
| Текст                      | TF-IDF бейзлайн / transformers fine-tuning             | [nlp.ipynb](nlp.ipynb)               |
| Любая задача               | KFold, ансамбли, pseudo-labeling, AMP                  | [tricks.ipynb](tricks.ipynb)         |


## Быстрый пайплайн по решению задачи

1. Прочитать данные, посмотреть `df.info()`, `df.describe()`, пропуски
2. Определить тип задачи: регрессия / классификация / кластеризация
3. Подумать о данных
4. Выбрать подход по таблице выше
5. Заюзать шаблон из нужного ноутбука
6. Если это не превый итерация задачки
  1. Задуматься об олимиадной хитрушечке
  2. Задуматься приемчике из [tricks.ipynb](tricks.ipynb)
7. Подставить свои данные, (подкрутить/подобрать гиперпараметры, начиная со второй итерации)
8. Сформировать submission.csv
9. Перейти к следующей задаче/ вернуться к 3 шагу
